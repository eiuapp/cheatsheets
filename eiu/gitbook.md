## 常用命令汇总

```
gitbook init bookname 创建书名为bookname的书
gitbook install 安装包
gitbook serve 本地预览
gitbook build src dest 编译src目录，生成到dest目录
gitbook help 查看命令帮助
```

### install gitbook ###

依赖于 nodejs

```bash
npm install -g gitbook gitbook-cli 
```
