### Expressions

```
Right Ctrl + F        -- 切换到全屏模式

Right Ctrl + L        -- 切换到无缝模式

Right Ctrl + C        -- 切换到比例模式

Right Ctrl + Home -- 显示控制菜单
```

### Change default shell

```bash
chsh -s `which zsh`
```
