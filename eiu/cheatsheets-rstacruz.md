### 如何在本地预览cheatsheet

```bash
which rvm && which ruby
cd cheatsheets #				进入目录
bundle install # https://mycyberuniverse.com/how-fix-bundler-command-not-found-jekyll.html
which nvm && which node
make -B _site
ls _site
npm i -g anywhere
which anywhere
anywhere -s # https://github.com/JacksonTian/anywhere/issues/35
```
